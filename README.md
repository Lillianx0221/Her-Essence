# Her Essence 灵境调酒屋

一个基于 WebGL (Three.js) 的交互式概念调酒项目。将个人性格推演与对历史上杰出女性的致敬相结合，通过 3D 视觉和色彩流动，生成独一无二的“性格祭酒”。

## 🌟 核心理念
- **致敬与记忆**：每一杯生成的酒，都对应一位在文学、科学、艺术等领域留下深刻印记的女性。
- **感官定制**：用户若不满意推演结果，可通过色值和气味描述词重新调配。
- **视觉沉浸**：采用 Three.js 构建真实的玻璃材质与流体填充动画。
